Application CallATaxi
Xiaojun CHENG et Benli JIN
20/10/14

*** 1/ Introduction
Ce projet est réaliser par CHENG Xiaojun et JIN Benli,l'application CallATaxi permet aux client de reserver un taxi sous leur portable, et également permet aux conducteur de taxi de recevoir les reservations de façon simple et efficace.


*** 2/ Utilisation d'application
Cette application est normalement fonctionné avec une BackOffice, nous avons mise une fichier .jar pour la BackOffice. Le détail de webservice est dans l'annexe.
Néanmoins, pour tester toutes les fonctionnalités de façon OFFLINE, c'est-t-dire lancer l'application sans WebService, nous avons mis en place des adaptations, par exemple pour le conducteur, on peut simuler de recevoir une reservation en cliquant le button (click button to simulate a client coming without web service).
Grace aux ces adaptation, vous pouvez tester toutes fonctionalité sans webservice.

*** 3/ Scénario d'usage
****** 3.1/ Utilisateur client 
	   Le client lancer l'application, l'application demander sous un voix : Welcome to the application, XXXXXXXX
	   Le client dit : Client
	   Le deuxieme écran s'afficher avec une carte, le client peut reserver un taxi dans le deuxième écran.
	   Une fois la reservation est validé par le serveur, le client va reçu un aperçu pour toutes les informations sur le taxi qui va lui prendre.
